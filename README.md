# Цифровой паспорт регионов Казахстана

Интерактивный desktop-first прототип аналитического дашборда на React, TypeScript и Tailwind CSS.

## Запуск

```bash
pnpm install
pnpm dev
```

Откройте адрес, указанный в терминале (обычно `http://localhost:3000`).

## Проверка production-сборки

```bash
pnpm build
pnpm start
```

Все числовые показатели в текущей версии помечены как демонстрационные, поскольку исходный PRD по пути `/mnt/data/prd.md` был недоступен в Windows-сессии.
